Technologies Used in the Project

Front-end:

  • React: A JavaScript library for building user interfaces.
  • Emotion: A CSS-in-JS library for styling React components.
  • Material-UI (MUI): A React component library for creating modern UIs.
  • Redux Toolkit: Tools for managing the state of your React application.
  • @stripe/react-stripe-js: Stripe library for handling payments.
  • Formik: A form library for handling form inputs in React.
  • React DnD: A library for drag-and-drop functionality.
  • React Image Gallery: A library for creating image galleries.
  • React Router DOM: A library for routing and navigation in React.

Back-end:

  • Node.js: A JavaScript runtime environment for server-side development.
  • Express: A web framework for Node.js.
  • Express Session: Middleware for managing sessions in Express.
  • Axios: A promise-based HTTP client for making requests.
  • Morgan: Middleware for HTTP request logging in Express.
  • CORS: A package for enabling cross-origin resource sharing.
  • Sequelize: An ORM for working with databases in Node.js.
  • Bcryptjs: Library for secure password hashing.
  • Stripe: Stripe API library for payment processing.

Database:

  • PostgreSQL: An open-source relational database management system.
  • pg and pg-hstore: PostgreSQL libraries for Node.js.

Other:

  • dotenv: Package for loading environment variables from a .env file.
  • image-thumbnail: Library for generating image thumbnails.
  • sonner: Custom or lesser-known library (provide a description if needed).
  • Vite-Express: Tool for building modern web applications with Vite and Express.
