Technologies Used in Project 2

Front-end:

  • @auth0/auth0-react: Authentication library for React applications.
  • @fortawesome/fontawesome-svg-core: Core library for working with FontAwesome icons in SVG format.
  • @fortawesome/free-brands-svg-icons: FontAwesome icons for various brands in SVG format.
  • @fortawesome/react-fontawesome: React wrapper for FontAwesome icons.
  • @headlessui/react: A set of completely unstyled, fully accessible UI components for React applications.
  • @stripe/react-stripe-js: Stripe library for handling payments.
  • @stripe/stripe-js: Stripe library for payment processing.
  • @tailwindcss/aspect-ratio: Tailwind CSS plugin for aspect ratio utilities.
  • @tailwindcss/forms: Tailwind CSS plugin for styling form elements.
  • @tailwindcss/typography: Tailwind CSS plugin for typography styles.
  • React Hook Form: A library for managing forms in React.
  • React Icons: A library for including popular icon sets in your React application.
  • React Router DOM: A library for routing and navigation in React.
  • Rsuite: A suite of React components for building rich and customizable applications.

Back-end:

  • Node.js: A JavaScript runtime environment for server-side development.
  • Express: A web framework for Node.js.
  • Express Session: Middleware for managing sessions in Express.
  • Axios: A promise-based HTTP client for making requests.
  • Bcrypt: A library for hashing passwords securely.
  • CORS: A package for enabling cross-origin resource sharing.
  • Crypto: A Node.js built-in module for cryptographic operations.
  • Dotenv: Package for loading environment variables from a .env file.
  • Email-Templates: A library for generating email templates.
  • Nodemailer: A library for sending email from Node.js applications.
  • PostgreSQL: An open-source relational database management system.
  • pg and pg-hstore: PostgreSQL libraries for Node.js.
  • Preview-Email: A library for previewing email templates.
  • Pug: A templating engine for generating HTML.
  • Sequelize: An ORM for working with databases in Node.js.
  • Stripe: Stripe API library for payment processing.
  • Sweetalert2: A JavaScript library for creating modals and pop-up dialogs.
  • Sweetalert2-React-Content: A library for rendering React components in Sweetalert2 modals.
  • tailwindcss-children: A Tailwind CSS plugin for styling child elements.
  • Vite-Express: Tool for building modern web applications with Vite and Express.
  • Vitest: A testing library for Vite.
